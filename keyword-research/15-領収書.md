## 분석

### 시장 현황
- **領収書**: 1만~10만 검색량 (**기준 충족**)
- 281개 키워드 아이디어 (중간 규모 시장)
- 인보이스 제도 관련 키워드 다수
- 입찰가 중~높음 (¥167~¥827) = 상업적 가치 있음

### 검색 의도 분석
1. **가이드 검색**: "書き方", "作り方" (작성 방법)
2. **인보이스 관련**: "インボイス対応", "インボイス領収書"
3. **온라인/디지털**: "オンライン", "ネット", "web"
4. **플랫폼 관련**: Amazon, 楽天, Shopify 영수증

### 레드오션 경고
**주요 경쟁사/플랫폼:**
- **Amazon**: 영수증 발행 자체 지원
- **楽天市場**: 영수증 발행 자체 지원
- **Google**: 영수증 템플릿 제공
- **LINE**: 디지털 영수증 서비스
- **Shopify**: EC 영수증 자동 발행

### 니치 기회
1. **領収書書き方 가이드**: 1만~10만 검색, 매우 낮은 입찰가 (¥2)
2. **インボイス対応領収書**: 인보이스 제도 대응 도구 (1천~1만)
3. **領収書オンライン 작성**: 온라인 영수증 작성 도구 (100~1천)
4. **領収書無料**: 무료 영수증 템플릿/도구 (100~1천)

### 리스크 요인
1. **높은 경쟁도**: 메인 키워드 경쟁 "높음"
2. **대기업 플랫폼 경쟁**: Amazon, 楽天, Shopify 자체 지원
3. **템플릿 포화**: 무료 템플릿 다수 존재
4. **見積書 시장과 중복**: 07-見積書.md에서 이미 레드오션 확인

### 결론
- **領収書**: 1만~10만 검색량 (**기준 충족**)
- 그러나 **見積書/請求書와 같은 문서 작성 시장**
- 大企業 플랫폼이 자체 영수증 발행 지원
- **제한적 기회**:
  - 領収書書き方 가이드 (정보성 콘텐츠, 낮은 경쟁)
  - インボイス対応 영수증 도구 (인보이스 제도 대응)
- **권장**: 見積書, 請求書, 領収書를 통합한 SaaS보다는 freee, マネーフォワード가 이미 지배
