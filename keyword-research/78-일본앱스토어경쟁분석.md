# 일본 앱스토어 경쟁 앱 현황 조사 (2025년)

## 조사 개요
- 조사 일자: 2025년 12월 25일
- 조사 대상: Google Play, Apple App Store (일본)
- 조사 카테고리: 5개 (작り置き/献立, 節約/家計簿, 子育て/ワーママ, 片付け/断捨離, 交換日記)

---

## 1. 作り置き / 献立アプリ (밑반찬/식단 앱)

### 1.1 주요 앱 현황

#### クラシル (kurashiru)
- **지위**: 국내 No.1 레시피 플랫폼 (DL수, 이용자수, 레시피 동영상수, SNS 팔로워수 모두 1위)
- **레시피 수**: 약 50,000건 이상의 동영상 레시피
- **특징**:
  - 관리영양사 감수
  - 전문 셰프가 제작한 레시피
  - 모든 레시피에 조리시간과 비용 표시
  - 실패 확률이 낮다는 평가
- **강점**: 프로가 만든 레시피로 신뢰성 높음

#### クックパッド (Cookpad)
- **이용자 수**: 월간 약 4,800만명
- **레시피 수**: 388만건 이상
- **과금**: 프리미엄 월 308엔
- **프리미엄 회원**: 130.7만명 (전년 대비 7.6% 감소, 2025년 Q3)

**주요 문제점**:
- 일반 사용자 투고 레시피로 품질 편차 큼
- "맛없는 레시피가 많아 진짜 원하는 레시피를 찾기 어렵다"
- "인기 레시ピ가 유료회원 전용이 되어 불만"
- 무료 사용자를 불편하게 만드는 유료화 방식
- 2017년 꿀 이유식 레시피 문제 (1세 미만 영아 위험)
- **실적 악화**: 프리미엄 회원 205만→170만명 미만, 주가 2,880엔→200엔(1/10), 이용자 1,000만명 감소

#### me:new (ミーニュー)
- **특징**: 최장 1주일분 식단 자동 제안
- **기능**: 식재료 목록 자동 작성, 관리영양사 감수
- **타겟**: 가족, 특히 부모와 자녀가 함께 즐길 수 있는 식사 제안
- **다운로드 수**: 구체적 수치 미확인 (검색 결과에 정보 없음)

#### Racook
- **조사 결과**: 검색 결과에서 구체적 정보 발견되지 않음
- 비교적 신규 서비스이거나 다른 이름으로 운영 중일 가능성

### 1.2 시장 특성

**레드오션 정도**: 높음
- 크라시루와 쿡패드가 시장 양분
- 다수의 레시피 앱 존재 (델리시키친, 기타)
- 차별화 전략 필수

**사용자 불만 사항**:
1. **레시피 품질**: 일반인 투고 레시피의 품질 편차
2. **과금 모델**: 인기 레시피가 유료 전용
3. **정보 과다**: "선택할 수 없는 고통" (쿡패드의 388만 레시피)
4. **안전성**: 건강/안전 관련 정보의 신뢰성 부족

**신규 진입 전략**:
- 특정 니치 타겟 (예: 워킹맘, 1인 가구, 절약)
- 레시피 품질 보증 시스템
- 합리적인 과금 모델
- 식단 자동화 + AI 추천

## 2. 節約 / 家計簿アプリ (절약/가계부 앱)

### 2.1 주요 앱 다운로드 수 & 평점

#### らくな家計簿
- **다운로드**: 2,000만 돌파
- **특징**: 간편함으로 정평

#### Zaim
- **다운로드**: 1,100만 이상
- **사용자 만족도**: 95.4%가 "지속하기 쉽다"고 응답
- **과금**: 무료로 연계 서비스 수 제한 없음
- **프리미엄 기능**: 이력 일괄 편집, 가계 개선 포인트 발견
- **학생 혜택**: Zaim 할인으로 1년 무료 (2025년 6월)
- **보안**:
  - 2048bit EV-SSL 암호화
  - 인증 정보는 별도 서버에 암호화 저장
  - 이체 비밀번호, 카드번호 등 미보관

#### シンプル家計簿 MoneyNote
- **다운로드**: 800만 돌파
- **성과**: App Store 무료 랭킹 1위

#### レシーピ！
- **다운로드**: 400만 돌파

#### Moneytree
- **다운로드**: 190만 돌파
- **특징**: 다양한 미디어 소개

#### マネーフォワード ME
- **지위**: 이용률 No.1 (2025년 10월)
- **보안**: 2048bit SSL 암호화, 데이터와 인증정보 별도 서버 저장

#### おカネレコ
- **특징**: 실행 즉시 입력 화면 (2초 가계부)

#### OsidOri (オシドリ)
- **특징**: 부부/커플 공유 가계부

### 2.2 부정적 리뷰 & 공통 불만점

#### 보안 우려
1. **정보 유출 위험**: 은행계좌, 신용카드 등 개인정보 취급
2. **불충분한 보안**: 로그인 ID/비밀번호 유출 시 부정 접근 위험
3. **위치 정보**: 영수증 레시토어 앱의 경우 주소 특정 가능
4. **서버 공격**: 사이버 공격으로 개인정보 탈취 가능성
5. **서비스 종료**: 갑작스런 서비스 종료 시 데이터 손실

#### 기능 제한
- 무료 사용 시 기능 제약이 큼
- 과금하지 않으면 사용 제한

#### 사용성
- 복잡한 기능에 대한 학습 곡선

### 2.3 시장 포화도

**레드오션 정도**: 매우 높음
- 다수의 확립된 플레이어 존재
- 천만 단위 다운로드 앱 다수

**차별화 포인트**:
- 보안 강화 메시지
- 틈새 타겟 (예: 식비 절약 특화)
- 심플한 UX
- 커플/가족 공유 기능

**신규 진입 가능성**: 중하
- 기존 앱들의 시장 지배력 강함
- 보안에 대한 사용자 우려가 높아 신규 앱 신뢰 구축 어려움
- 특정 니치 시장 공략 또는 혁신적 기능 필요

## 3. 子育て / ワーママアプリ (육아/워킹맘 앱)

### 3.1 주요 앱 현황

#### ninaru 시리즈
- **총 다운로드**: 누적 1,000만 돌파
- **평가**: iOS/Android 모두 4.5 이상
- **시장 점유**: 임산부 4명 중 1명 사용
- **MAU**: 2025년 10월 과거 최고 기록 (전년 동월 대비 103.9% 증가)
- **특징**:
  - 소아과의사/조산사 감수
  - 아이 성장에 맞춘 메시지와 기사 (7세까지)
  - 이용자 평가 No.1

#### ninaru baby
- **시장 점유**: 엄마 3명 중 1명 사용
- **온라인 교실**: 누적 15만명 참가, 만족도 98%

**주요 불만 사항**:
1. **날씨 예보 정확도**: 실제 기온과 차이 큼
2. **기사 내용 편향**: 성장이 빠른 아이 위주로 소개되어 비교하게 됨
3. **조작성**: 형제자매 전환이 번거로움, 탭 횟수 많음
4. **앱 불안정성**: 강제 종료
5. **광고**: 개인정보 유출 우려, 광고량 과다
6. **신뢰성**: 근거 부족하거나 불안을 조장하는 기사
7. **기능 제약**: 이유식 시작 시기 직접 설정 불가
8. **지역 설정**: "해당 지역 없음" 오류
9. **복수 자녀**: 연년생 육아 시 각각 등록 불가
10. **아빠 배려 부족**: 엄마와 아이 중심, 파트너/가족에 대한 조언 부족

#### ママリ (Mamari)
- **이용자**: 2018년 출산한 엄마 3명 중 1명
- **월간 조회수**: 약 2억회
- **월간 투고**: 130만건 이상
- **서비스 시작**: 2014년
- **특징**: 임신/출산/육아 Q&A 커뮤니티

### 3.2 시장 포화도

**레드오션 정도**: 높음
- ninaru와 ママリ가 시장 양분
- 각각 수백만~천만 사용자 확보
- 출생률 감소 속에서도 MAU 증가 (시장 집중화)

**차별화 포인트**:
- 워킹맘 특화 기능
- 아빠/가족 참여 기능
- 정확한 정보 제공 (의학적 근거)
- 형제자매 동시 관리
- 심플한 UI/UX

**신규 진입 가능성**: 중하
- 기존 플레이어의 강력한 브랜드 파워
- 의료 감수 등 콘텐츠 생산 비용 높음
- 틈새 시장 공략 필요 (예: 워킹맘 전용, 아빠 육아, 쌍둥이/다자녀)

## 4. 片付け / 断捨離アプリ (정리정돈 앱)

### 4.1 주요 앱 현황

#### みん断・一日一捨
- **지위**: 단샤리 앱 ASO 검색 1위
- **미디어**: 2025년 3월 일본TV "DayDay." 소개
- **특징**:
  - 로그인 불필요, 익명 참여
  - SNS형 단샤리 기록 앱
  - 타인의 "버린 보고" 확인으로 자극
  - 물건뿐 아니라 마음의 정리도 가능
  - 버린 개수 자동 카운트
  - 월간 30개 목표 설정
  - 계속 일수 기록
  - 개인정보 검색 불가, 댓글 기능 없음 (비방 걱정 없음)

**사용자 평가**:
- "명확한 기능, 심플한 화면, 간단한 조작"
- "버린 개수 카운터로 동기부여"
- "다른 사람 투고로 자극받음"

#### 1SSHA - 一日一捨・断捨離アプリ
- **컨셉**: 하루에 하나씩 버리기
- **기능**:
  - 버린 것/판 것 등 하루에 하나 입력
  - 캘린더에 기록
  - 공개하면 타인 열람 가능
  - 귀여운 고양이 캐릭터
  - 캘린더 색 변화로 동기부여

**프리미엄 기능**:
- 투고 시 이미지 설정 가능
- 1일 2개까지 투고
- 광고 비표시
- 월별 일람 기능
- 영구 구매 플랜 제공

**사용자 평가**:
- "다른 단샤리 앱보다 디자인이 심플하고 좋다"
- "삼일천하인 나도 계속할 수 있을 것 같다"

#### 기타 경쟁 앱
- **KonMari Spark Joy!**: 곤도 마리에 메소드 게임 (유료: App Store 600엔/Google Play 490엔)
- **Let it GO! online**: Google Play에서 확인 가능

### 4.2 시장 특성

**레드오션 정도**: 중
- 다수의 단샤리 앱 존재
- 무료/유료 혼재
- 명확한 1위 없음 (みん断이 ASO 검색 1위)

**경쟁 강도**: 중
- 기능이 비슷한 앱들 다수
- 디자인과 UX로 차별화
- 커뮤니티 기능 유무
- 유료화 모델 차이

**공통 불만점**:
- 검색 결과에 구체적 불만 사항 없음
- 비교적 만족도 높은 편

**신규 진입 가능성**: 중상
- 시장 성숙도가 낮음
- 기능 차별화 여지 존재
- 심리적 동기부여가 핵심
- 게이미피케이션 요소 추가 가능
- SNS 연계 강화 여지

**차별화 포인트**:
- AI 기반 정리 조언
- 버린 물건의 탄소 절감 계산
- 중고 거래 플랫폼 연계
- 가족/친구와 경쟁 기능
- 정리 전후 사진 비교

## 5. 交換日記アプリ (교환일기 앱)

### 5.1 주요 앱 현황

#### Shanfy (シャンフィー)
- **컨셉**: 일기 × SNS 교환일기
- **기능**:
  - 매일 유니크한 질문 제공
  - "복권 당첨되면?" 같은 질문에 답변
  - 멤버들이 순서대로 답변 투고
  - 텍스트 전용 (심플 설계)
  - 최대 4장 사진 첨부
  - 비공개 설정 가능
  - 캘린더 형식 표시
  - PDF 저장 가능

**대대적 리뉴얼 논란** (2024년 10월경):
- 기존 교환일기 기능 폐지
- "10/31에 데이터 전부 삭제, 기존 일기는 1,800엔에 구매 가능" 공지
- **사용자 강력 반발**:
  - "3년간 사용, 일기 3,000개 가까이 저장했는데 갑작스런 공지 없이 삭제"
  - "광고 넣거나 광고 제거 유료판 팔면 됐을 것, 이렇게 하니 인상 나쁘다"
  - "다시는 사용 안 함, 다른 교환일기 앱으로 이동"
  - "3-4년 사용, 친구와 서로 일기 읽고 사진도 올리고 좋았는데..."
  - "너무 갑작스럽고 슬프다"
  - "1,800엔 울며 겨자먹기로 지불했지만 PDF는 본인 일기만 가능"
  - "새 버전은 완전히 다른 앱, 전혀 쓰고 싶지 않다"
  - "최소한 이전 버전처럼 교환일기 할 수 있는 앱으로 되돌려 달라"

#### 記憶の木
- **타겟**: 커플 전용
- **기능**:
  - 매일 1문 동일 질문
  - 2명 각각 답변 기록
  - 답변 완료 시 숲에 물 주기, 나무/꽃 성장
  - 사진 첨부 가능
  - "두 사람의 숲" 성장 = 추억의 상징
  - 커플 게임, 타임캡슐, 힐링 음악
- **과금**: 1명이 프리미엄 구매하면 2명 사용 가능

**평가**:
- 동일 질문으로 서로의 마음 이해
- 놀이 요소 풍부

#### Between (ビトウィーン)
- **개발**: 한국 VCNC사
- **타겟**: 커플 전용
- **기능**: 메시지, 사진, 기념일 관리 등 2명만의 프라이빗 공간
- **다운로드**: 전세계 수천만 (구체적 수치 미확인)
- **플랫폼**: iOS, Android

**조사 한계**: 검색 결과에 구체적 정보 없음

### 5.2 시장 특성

**레드오션 정도**: 중하
- 교환일기 앱 다수 존재하지만 천만 단위 앱은 없음
- Shanfy의 리뉴얼 실패로 사용자 이탈 가능성

**2025년 트렌드**:
- 가족, 커플, 친구 간 커뮤니케이션 활성화
- 무료 + 광고 없음 선호
- 심플한 조작성 중시

**Shanfy 이탈 관련 리뷰 핵심**:
1. **사전 공지 부족**: 갑작스런 서비스 변경
2. **데이터 유실**: 수년간의 추억 삭제
3. **과도한 과금**: 기존 데이터 보존에 1,800엔
4. **기능 변경**: 교환일기 기능 완전 제거
5. **신뢰 상실**: "다시는 사용하지 않겠다"는 반응 다수

**공통 불만점**:
- 서비스 안정성/지속성에 대한 불안
- 데이터 소유권 문제
- 과금 모델 불만
- 기능 변경에 대한 사전 소통 부족

**신규 진입 가능성**: 중상
- Shanfy 이탈 사용자 확보 기회
- 시장이 완전 포화되지 않음
- 차별화된 기능으로 승부 가능

**차별화 포인트**:
- 데이터 영구 보증 (사용자 소유권)
- 투명한 과금 모델
- 다양한 형태 지원 (커플/가족/친구/모르는 사람)
- AI 질문 생성
- 장기 보관 및 책 제작 서비스
- 감정 분석 및 관계 인사이트

### 신규 진입 전략 제안

#### 높은 잠재력 카테고리

**1순위: 片付け/断捨離 (정리정돈)**
- 시장 성숙도 낮음
- 기능 차별화 여지 큼
- 심리/게이미피케이션 요소 강화 가능
- 환경(탄소 절감), 경제(중고거래) 연계 가능

**2순위: 交換日記 (교환일기)**
- Shanfy 이탈자 흡수 기회
- 데이터 안정성/영구 보증으로 신뢰 확보
- 다양한 관계 유형 지원 (가족/친구/커플/랜덤)
- AI 질문 생성으로 차별화

#### 블루오션 전략 (기존 카테고리 재해석)

**3순위: 작り置き/献立 × 절약/가계부 융합**
- 식비 절약에 특화된 식단 앱
- 가계부와 연동하여 식비 자동 분석
- 냉장고 재고 관리 + 유통기한 알림
- 저렴한 식재료 기반 레시피 추천
- 워킹맘 타겟: 주말 작り置き 특화

**4순위: 子育て/ワーママ 틈새 시장**
- 아빠 육아 특화 (기존 앱의 약점)
- 다자녀/쌍둥이 가정 특화
- 워킹맘의 시간 관리 + 육아 통합
- 가족 전체 참여형 육아 앱

### 공통 성공 요소

1. **신뢰성**: 의학/전문가 감수, 데이터 보안
2. **심플 UX**: 3초 이내 핵심 기능 접근
3. **합리적 과금**: 투명하고 공정한 유료화 모델
4. **커뮤니티**: 동기부여와 정보 공유
5. **데이터 소유권**: 사용자 데이터 영구 보증
6. **차별화 기능**: 기존 앱에 없는 1가지 킬러 기능

### 피해야 할 실수 (기존 앱 사례)

1. **쿡패드**: 무료 사용자를 불편하게 만드는 유료화
2. **Shanfy**: 사전 공지 없는 대대적 서비스 변경, 데이터 삭제
3. **가계부 앱**: 보안에 대한 명확한 메시지 부재
4. **ninaru baby**: 기사 편향, 파트너 배려 부족, 광고 과다
5. **공통**: 정보 과다로 인한 선택 장애
