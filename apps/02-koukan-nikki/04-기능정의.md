## 1. MVP 기능 (v1.0)

### 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| **일기장 생성** | 커플/가족/친구 일기장 | P0 |
| **일기 작성** | 텍스트 + 사진 | P0 |
| **푸시 알림** | 새 글 알림 | P0 |
| **읽음 확인** | 상대방 읽음 표시 | P0 |
| **자동 백업** | 무료 클라우드 백업 | P0 |
| **내보내기** | PDF/이미지 | P1 |

## 3. 확장 기능 (v2.0)

### 3.1 AI 질문 생성

```
오늘의 질문 (AI 추천)

"최근 상대방에게 고마웠던 순간은?"

[이 질문으로 시작]  [다른 질문]
```

### 3.2 손글씨 입력 (프리미엄)

- 손가락/펜으로 직접 쓰기
- 손글씨 폰트 변환
- 원본 손글씨 저장

### 3.3 음성 메모 (프리미엄)

- 음성 녹음 첨부
- 텍스트 변환 (선택)

### 3.4 1년 전 오늘

```
📅 1년 전 오늘의 일기

"작년 크리스마스에는..."

[보러 가기]
```

### 3.5 기념일 자동 기록

```
🎉 특별한 날 알림

오늘은 사귄 지 100일째!
첫 일기를 쓴 지 1년!

[축하 일기 쓰기]
```

## 4. 수익화 기능

### 무료 (Free)

- 1개 일기장
- 기본 일기 작성
- 자동 백업
- 푸시 알림
- 광고 표시

### 프리미엄 (월 380엔)

- 무제한 일기장
- 손글씨 입력
- 음성 메모
- 동영상 첨부
- 광고 제거
- 프리미엄 스티커

### 커플 플랜 (월 580엔)

- 2명 프리미엄
- 기념일 캘린더
- 커플 전용 테마

## 5. 기술 요구사항

### 프론트엔드

```
React Native
├── 화면
│   ├── 홈 (일기장 목록)
│   ├── 일기장 상세
│   ├── 일기 작성
│   ├── 설정
│   ├── 데이터 안전 센터
│   └── 프리미엄
├── 컴포넌트
│   ├── 일기 카드
│   ├── 에디터
│   ├── 사진 첨부
│   └── 스티커
└── 상태관리
    └── Zustand / Redux
```

### 백엔드

```
Supabase
├── Auth (소셜 로그인 + 이메일)
├── Database
│   ├── users
│   ├── diaries (일기장)
│   ├── entries (일기)
│   ├── members (멤버)
│   └── notifications
├── Storage (이미지, 음성)
├── Edge Functions
│   ├── 푸시 알림
│   └── AI 질문 생성
└── Realtime (읽음 확인)
```

### 푸시 알림

```
Firebase Cloud Messaging (FCM)
├── 새 일기 알림
├── 읽음 확인
├── 리마인더
└── 기념일 알림
```

## 7. 개발 로드맵

### Phase 1: MVP (5주)

| 주차 | 작업 |
|------|------|
| 1주 | 프로젝트 셋업, Auth, DB 설계 |
| 2주 | 일기장 생성, 멤버 초대 |
| 3주 | 일기 작성, 사진 첨부 |
| 4주 | 푸시 알림, 읽음 확인 |
| 5주 | 테스트, 백업, 출시 준비 |

### Phase 2: 확장 (4주)

| 주차 | 작업 |
|------|------|
| 6주 | AI 질문 생성 |
| 7주 | 손글씨 입력 |
| 8주 | 음성 메모 |
| 9주 | 기념일, 1년 전 오늘 |

### Phase 3: 수익화 (3주)

| 주차 | 작업 |
|------|------|
| 10주 | 프리미엄 결제 연동 |
| 11주 | 광고 연동 |
| 12주 | 마케팅, 정식 출시 |
