# 交換日記 (코우칸닛키) - 교환일기 앱

> **프로젝트 상태**: 기획 단계
> **우선순위**: 2순위 (Shanfy 이탈자 흡수 기회)
> **플랫폼**: 모바일 앱 (React Native)

---

## 프로젝트 개요

일본 시장을 타겟으로 한 교환일기 앱. Shanfy의 대규모 리뉴얼 실패로 이탈한 사용자를 흡수하고, "데이터 안전성"을 핵심 가치로 차별화.

### 핵심 포지셔닝

```
"모든 관계를 위한 신뢰할 수 있는 교환일기"

- Shanfy 이탈 유저 흡수
- 데이터 안전성 강조 (절대 삭제 안 함)
- 커플+가족+친구 통합 지원
```

---

## 시장 기회

| 지표 | 수치 |
|------|------|
| 검색량 | 1만~10만/월 (+900% 성장) |
| 경쟁도 | 낮음 |
| 앱스토어 경쟁 | Shanfy 이탈 중, 명확한 대안 없음 |
| 진입 가능성 | **중상** |

### 특별 기회: Shanfy 이탈

```
2024년 10월 Shanfy 대규모 리뉴얼 실패:
- 3-4년치 일기 데이터 삭제 예고
- 기존 데이터 백업에 1,800엔 요구
- "다시는 사용 안 함" 리뷰 폭주
→ 이탈 유저 흡수 절호의 기회!
```
