## 3. USP (Unique Selling Proposition)

### 1차 USP: 데이터 영구 보증

```
"절대 삭제하지 않습니다"

✅ 자동 백업 (무료)
✅ PDF/이미지 내보내기
✅ 로컬 저장 옵션
✅ 서비스 종료 시에도 데이터 보장 약속
```

### 2차 USP: 다중 관계 지원

```
"모든 관계를 위한 교환일기"

📕 커플 일기장
📗 가족 일기장
📘 친구 그룹 일기장
📙 익명 (코드로 연결)
```

### 3차 USP: 100% 푸시 알림

```
"상대방 글, 절대 놓치지 않아요"

✅ 새 글 즉시 알림
✅ 읽음 확인 표시
✅ 작성 리마인더
```
