## 2. 주요 경쟁 앱

### Shanfy (シャンフィー) - 이탈 중

| 항목 | 내용 |
|------|------|
| **상태** | 대규모 리뉴얼 후 사용자 이탈 중 |
| **평점** | 급락 (리뉴얼 후 부정 리뷰 폭주) |

**기존 기능** (폐지됨):
- 매일 유니크한 질문 제공
- 멤버들이 순서대로 답변
- 텍스트 전용 (심플)
- 최대 4장 사진 첨부
- 캘린더 형식 표시
- PDF 저장

**리뉴얼 논란** (2024년 10월):
```
❌ 기존 교환일기 기능 폐지
❌ "10/31에 데이터 전부 삭제" 공지
❌ 기존 일기 백업 1,800엔에 판매
❌ 사전 공지 부족
```

**사용자 반응**:
- "3년간 사용, 일기 3,000개 갑자기 삭제"
- "광고 넣거나 유료판 팔면 됐을 것, 인상 나쁘다"
- "다시는 사용 안 함"
- "1,800엔 울며 겨자먹기로 지불, PDF는 본인 일기만"

### 記憶の木 - 커플 전용

| 항목 | 내용 |
|------|------|
| **타겟** | 커플 전용 |
| **특징** | 질문 기반, 게이미피케이션 |

**주요 기능**:
- 매일 1문 동일 질문
- 2명 각각 답변 기록
- 답변 완료 시 숲에 물 주기
- 나무/꽃 성장 (추억의 상징)
- 사진 첨부
- 커플 게임, 타임캡슐, 힐링 음악

**과금**:
- 1명이 프리미엄 구매 시 2명 사용 가능

**한계점**:
- **커플만 가능** (친구/가족 X)
- 그룹 지원 없음
