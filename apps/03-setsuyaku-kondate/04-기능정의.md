## 5. 기술 요구사항

### 프론트엔드 (Kimatch 포크)

```
Next.js
├── 페이지
│   ├── 홈 (대시보드)
│   ├── 식단 플래너
│   ├── 레시피 목록/상세
│   ├── 장보기 리스트
│   ├── 절약 리포트
│   └── 설정
├── 컴포넌트
│   ├── 예산 슬라이더
│   ├── 캘린더
│   ├── 타임라인
│   └── 차트
└── PWA
    ├── 오프라인 저장
    └── 푸시 알림
```

### 백엔드

```
Next.js API Routes + Prisma
├── Auth (NextAuth.js)
├── Database (PostgreSQL)
│   ├── users
│   ├── recipes
│   ├── meal_plans
│   ├── shopping_lists
│   └── budgets
├── AI (OpenAI API)
│   ├── 식단 생성
│   └── 레시피 추천
└── 외부 연동
    ├── 특매 정보 API
    └── 결제 (Stripe)
```
