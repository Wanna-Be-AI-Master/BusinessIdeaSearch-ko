## 1. 앱스토어 경쟁 현황

## 2. 주요 앱 경쟁사

### クラシル (kurashiru)

| 항목 | 내용 |
|------|------|
| **지위** | 국내 No.1 레시피 플랫폼 |
| **레시피 수** | 약 50,000건 이상 동영상 |
| **과금** | 프리미엄 월 480엔 |

**강점**:
- 관리영양사 감수
- 전문 셰프가 제작한 레시피
- 모든 레시피에 조리시간/비용 표시
- 실패 확률 낮음

**약점**:
- 작り置き "전문" 아님
- 절약 "전문" 아님

---

### クックパッド (Cookpad)

| 항목 | 내용 |
|------|------|
| **이용자** | 월간 약 4,800만명 |
| **레시피 수** | 388만건 이상 |
| **과금** | 프리미엄 월 308엔 |

**문제점**:
- 일반 사용자 투고 → 품질 편차
- 프리미엄 회원 감소 (205만→170만)
- 주가 2,880엔→200엔 폭락
- 사용자 1,000만명 감소

**사용자 불만**:
- "맛없는 레시피가 많아 진짜 원하는 레시피를 찾기 어렵다"
- "인기 레시피가 유료 전용"
- 2017년 꿀 이유식 레시피 문제

---

### me:new (ミーニュー)

| 항목 | 내용 |
|------|------|
| **특징** | 1주일 식단 자동 생성 |
| **다운로드** | 220만 DL |
| **평점** | App Store 4.3점 |

**사용자 불만**:
- 메뉴 바리에이션 부족
- 검색/동작 속도 느림
- 인원수 유연 설정 불가
- 헌립 입れ替え 불가능
- 가족 공유 기능 없음

## 3. 웹 경쟁 현황

### 시장 상황: 블루오션

- 블로그 위주, 전문 웹앱 부재
- つくりおき食堂이 압도적이지만 "블로그"

---

### つくりおき食堂 (마리에고항)

| 항목 | 내용 |
|------|------|
| **월간 PV** | 500만 |
| **누적 PV** | 2.3억 |
| **서적 판매** | 15만부 |
| **형태** | **블로그 (앱 없음!)** |

**특징**:
- 작り置き 전문
- 레시피 품질 높음
- 개인 블로그

**기회**:
- 앱/웹앱 없음 → "つくりおき食堂의 앱 버전" 포지셔닝
- 블로그 형태의 한계 (검색, 저장, 플래닝)
