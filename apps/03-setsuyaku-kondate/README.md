# 節約献立 + 作り置き (세츠야쿠 콘다테) - 식비 절약 밑반찬 앱

> **프로젝트 상태**: 기획 단계
> **우선순위**: 3순위 (앱은 레드오션, 웹은 기회)
> **플랫폼**: 웹앱 (Next.js + PWA)

---

## 프로젝트 개요

일본 시장을 타겟으로 한 식비 절약 + 밑반찬 통합 웹앱. 앱스토어는 クラシル/クックパッド가 독점하고 있으나, 웹에서는 つくりおき食堂(월 500만 PV) 같은 블로그가 성공. 웹앱으로 SEO 트래픽 공략.

### 핵심 포지셔닝

```
"월 식비 2만엔으로 건강하게 먹는 앱"

- 예산 먼저 설정 → 식단 역산
- 1주일 작り置き 전문
- つくりおき食堂의 "앱 버전"
```

---

## 시장 기회

| 지표 | 수치 |
|------|------|
| 節約 검색량 | 10만~100만/월 (+900% 성장) |
| 作り置き 검색량 | 1만~10만/월 |
| 앱 경쟁도 | **높음** (クラシル, クックパッド) |
| 웹 경쟁도 | **낮음** (블로그 위주) |

### 왜 웹인가?

```
앱스토어: 레드오션
├── クラシル: 국내 1위, 5만+ 레시피
├── クックパッド: 388만 레시피
└── → 직접 경쟁 어려움

웹: 블루오션
├── つくりおき食堂: 월 500만 PV (블로그)
├── 전문 웹앱 부재
└── → SEO로 트래픽 확보 가능
```
