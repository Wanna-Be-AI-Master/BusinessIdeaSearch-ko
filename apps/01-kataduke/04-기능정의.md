# 片付け 기능정의

> 조사일: 2025-12-25

## 1. MVP 기능 (v1.0)

### 핵심 기능

| 기능 | 설명 | 우선순위 |
|------|------|----------|
| **일일 정리 트래커** | 사진 + 메모로 버린 것 기록 | P0 |
| **카테고리 분류** | 의류/서적/잡화 등 자동 분류 | P0 |
| **연속 일수 카운터** | 연속 정리 일수 표시 | P0 |
| **월간 통계** | 이번 달 정리 개수, 카테고리별 | P1 |
| **리마인더** | 매일 알림 (시간 설정) | P1 |

## 2. 기능 상세

### 2.1 일일 정리 트래커

```
[사진 촬영/선택]
     ↓
[카테고리 선택] - 의류, 서적, 잡화, 가전, 기타
     ↓
[버린 이유 선택] - 사용 안함, 고장, 유행 지남, 충동구매, 기타
     ↓
[처리 방법] - 버림, 판매, 기부, 재활용
     ↓
[저장]
```

**UI 요구사항**:
- 3탭 이내 완료
- 사진 촬영 즉시 가능
- 카테고리/이유는 아이콘으로 빠른 선택

### 2.2 대시보드

```
┌─────────────────────────────────┐
│  오늘의 정리: 2개               │
│  연속: 7일 🔥                   │
│  이번 달: 23개                  │
├─────────────────────────────────┤
│  [카테고리별 차트]              │
│  의류 40% | 서적 30% | 잡화 30% │
├─────────────────────────────────┤
│  [최근 정리 목록]               │
│  - 청바지 (의류) 12/24          │
│  - 소설책 (서적) 12/24          │
└─────────────────────────────────┘
```

### 2.3 게이미피케이션 (v1.0)

| 요소 | 설명 |
|------|------|
| **연속 일수** | 매일 정리 시 카운트 증가 |
| **주간 목표** | 7개/주 목표 설정 |
| **뱃지** | 7일 연속, 30일 연속, 100개 달성 등 |

## 3. 확장 기능 (v2.0)

### 3.1 AI 버림 분석

```
"이번 달 인사이트"

📊 가장 많이 버린 카테고리: 의류 (15개)
💡 충동구매 의류가 60%를 차지합니다
🎯 다음 달 의류 구매 전 24시간 대기 규칙을 추천합니다
```

### 3.2 Before/After

- 방 사진 촬영 (정리 전)
- 정리 후 사진 촬영
- 비교 슬라이더 UI
- 타임라인 저장

### 3.3 환경 영향 계산

```
🌍 이번 달 환경 기여

- 정리한 물건: 23개
- CO2 절감: 12.5kg
- 나무 1그루 심은 효과!

[배지: 에코 히어로 획득!]
```

### 3.4 メルカリ 연동

```
"이 물건, 팔아보세요!"

👕 청바지 (ユニクロ)
💰 예상 판매가: ¥1,500 - ¥2,500
📦 판매 중인 유사 상품: 234개

[メルカリで出品する]
```

## 5. 기술 요구사항

### 프론트엔드

```
React Native
├── 화면
│   ├── 홈 (대시보드)
│   ├── 기록 (트래커)
│   ├── 통계
│   ├── 설정
│   └── 프리미엄
├── 컴포넌트
│   ├── 카메라
│   ├── 카테고리 선택기
│   ├── 차트
│   └── 뱃지
└── 상태관리
    └── Zustand / Redux
```

### 백엔드

```
Supabase / Firebase
├── Auth (소셜 로그인)
├── Database
│   ├── users
│   ├── items (정리 기록)
│   ├── categories
│   └── achievements
├── Storage (이미지)
└── Functions
    ├── AI 분석
    └── 통계 계산
```

### 외부 연동

| 서비스 | 용도 | 우선순위 |
|--------|------|----------|
| OpenAI API | AI 분석 | v2.0 |
| メルカリ API | 출품 연동 | v2.0 |
| Firebase Analytics | 사용자 분석 | v1.0 |
