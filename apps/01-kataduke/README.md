# 片付け (카타즈케) - 정리정돈 앱

> **프로젝트 상태**: 기획 단계
> **우선순위**: 1순위 (블루오션)
> **플랫폼**: 모바일 앱 (React Native)

---

## 프로젝트 개요

일본 시장을 타겟으로 한 정리정돈/단샤리 앱. 기존 앱들의 불만점을 해결하고 AI + 게이미피케이션으로 차별화.

### 핵심 포지셔닝

```
"물건을 줄이고, 삶을 늘리는 앱"
- SNS 기능 최소화 (선택적)
- 버린 이유 분석 → 소비 습관 개선 연계
```

---

## 시장 기회

| 지표 | 수치 |
|------|------|
| 검색량 | 1만~10만/월 |
| 경쟁도 | 낮음 |
| 앱스토어 경쟁 | 명확한 1위 없음 |
| 진입 가능성 | **중상** |
