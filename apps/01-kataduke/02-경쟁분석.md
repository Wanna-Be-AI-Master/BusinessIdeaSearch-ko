## 2. 주요 경쟁 앱

### みん断・一日一捨

| 항목 | 내용 |
|------|------|
| **지위** | 단샤리 앱 ASO 검색 1위 |
| **미디어** | 2025년 3월 일본TV "DayDay." 소개 |
| **다운로드** | 비공개 (대형 앱 아님) |

**주요 기능**:
- 로그인 불필요, 익명 참여
- SNS형 단샤리 기록
- 타인의 "버린 보고" 확인
- 버린 개수 자동 카운트
- 월간 30개 목표 설정
- 계속 일수 기록

**사용자 평가**:
- "명확한 기능, 심플한 화면, 간단한 조작"
- "버린 개수 카운터로 동기부여"
- "다른 사람 투고로 자극받음"

**한계점**:
- 동작 느림
- "왜 버렸는지" 분석 없음
- 개인화 부족

### 1SSHA - 一日一捨・断捨離アプリ

| 항목 | 내용 |
|------|------|
| **지위** | 신생 앱 |
| **특징** | 심플 디자인, 귀여운 UI |

**주요 기능**:
- 하루에 하나씩 버리기 기록
- 캘린더에 기록
- 공개 설정 가능
- 귀여운 고양이 캐릭터
- 캘린더 색 변화로 동기부여

**프리미엄 기능**:
- 투고 시 이미지 설정
- 1일 2개까지 투고
- 광고 제거
- 월별 일람
- 영구 구매 플랜

**사용자 평가**:
- "다른 단샤리 앱보다 디자인이 심플하고 좋다"
- "삼일천하인 나도 계속할 수 있을 것 같다"

## 3. 경쟁사 공통 불만점

| 불만 | 설명 | 기회 |
|------|------|------|
| **SNS 기능 불필요** | "남의 버린 물건 관심 없다" | 프라이버시 모드 |
| **분석 없음** | "왜 버렸는지" 인사이트 없음 | AI 분석 |
| **동기부여 한계** | 연속 일수 보상 없음 | 게이미피케이션 |
| **실용 도움 부족** | 수납 솔루션 없음 | 실용 도구 연계 |

## 4. 경쟁 우위 확보 전략

### 차별화 포인트

1. **AI 정리 조언**: "이 카테고리를 많이 버리네요. 충동구매 주의!"
2. **환경 연계**: 버린 물건의 탄소 절감량 계산
3. **중고 거래 연동**: メルカリ 원클릭 출품
4. **Before/After**: 방 사진 비교 시각화
5. **게이미피케이션**: 레벨, 뱃지, 챌린지
