## 2. 상세 보류 사유

### 2.1 경쟁사 독점

```
ninaru 시리즈: 1,000만 다운로드
ママリ: 600만 MAU, 월 2억 조회

→ 양강 구도 완성
→ 신규 진입 극도로 어려움
```

### 2.2 높은 콘텐츠 비용

| 필요 콘텐츠 | 비용 요인 |
|------------|----------|
| 육아 정보 | 소아과의사/조산사 감수 필요 |
| 건강 정보 | 의학적 정확성 필수 |
| 발달 가이드 | 전문가 콘텐츠 |

→ 개인/소규모 팀이 감당하기 어려움

### 2.3 네트워크 효과

```
ママリ 커뮤니티:
- 월 130만건 투고
- Q&A 형태 → 사용자가 사용자를 끌어옴
- 후발 주자가 따라잡기 어려움
```

### 2.4 수익화 어려움

- 육아앱은 무료 기대 높음
- 프리미엄 전환률 낮음
- 광고 의존 → 사용자 반발

## 4. 재검토 조건

### 언제 다시 검토할까?

1. **경쟁사 실수**
   - Shanfy처럼 대규모 리뉴얼 실패
   - 서비스 종료
   - 신뢰 상실 사건

2. **틈새 시장 발견**
   - 워킹대디 시장 급성장
   - 다자녀 가정 전용 니즈 확인

3. **B2B 기회**
   - 기업에서 복지 프로그램 문의
   - 기업 계약 기회

4. **기술 혁신**
   - AI가 콘텐츠 비용 대폭 절감
   - 차별화 기능 개발
