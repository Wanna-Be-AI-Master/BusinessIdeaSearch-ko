# 子育て両立 (코소다테 료우리츠) - 일-육아 양립 앱

> **프로젝트 상태**: ⚠️ **보류**
> **우선순위**: 4순위 (레드오션)
> **보류 사유**: ninaru + ママリ 양강 구도, 직접 경쟁 어려움

---

## 프로젝트 개요

일본 시장을 타겟으로 한 일-육아 양립 지원 앱. 워킹맘/워킹대디의 92%가 "양립 곤란"을 느끼는 시장이지만, 이미 ninaru(1,000만 DL)와 ママリ(600만 MAU)가 독점.

### 핵심 포지셔닝 (검토 중)

```
"워킹맘/대디의 일-육아 양립 전용 앱"

- 기존 육아앱 + 캘린더/업무 도구 융합
- 92%가 느끼는 "양립 곤란" 직접 해결
```

---

## ⚠️ 보류 사유

### 시장 상황: 레드오션

| 앱 | 규모 | 특징 |
|-----|------|------|
| **ninaru** | 1,000만 DL | 임산부 4명 중 1명 사용 |
| **ママリ** | 600만 MAU, 월 2억 조회 | Q&A 커뮤니티 독점 |

### 진입 장벽

1. **브랜드 파워**: 기존 앱의 강력한 인지도
2. **콘텐츠 비용**: 의료/육아 정보 전문가 감수 필요
3. **네트워크 효과**: 커뮤니티 앱의 특성상 사용자 확보 어려움

---

## 틈새 시장 가능성 (향후 검토)

### 1. 워킹대디 특화

```
기존 앱 불만: "아빠 배려 부족, 엄마와 아이 중심"

→ 아빠 육아 전용 앱으로 차별화?
```

### 2. 쌍둥이/다자녀 특화

```
기존 앱 불만: "복수 자녀 등록 불가"

→ 다자녀 가정 전용?
```

### 3. B2B 기업 복지

```
기업 복지 프로그램:
- 회사에서 직원에게 제공
- 기업 계약으로 수익화

→ B2C가 아닌 B2B 접근?
```
